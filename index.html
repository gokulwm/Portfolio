Pace Travels

Online Travel Booking & Fleet Management System

In Progress

🎯 Project Goal

A web-based travel booking platform for a small travel business enabling customers to book round-trip journeys, view estimated pricing, and receive booking confirmation after owner approval. Manages vehicles, drivers, trip details, and provides separate dashboards for customers, drivers, and the owner.

⚠️ Problem Statement

Small travel businesses rely on manual calls and messaging apps for booking, leading to miscommunication, double bookings, and no centralized fleet or driver management. Pace Travels digitizes the entire workflow from inquiry to trip completion.

Disaster Relief Coordination

Emergency Response & Resource Navigation System

Planning Phase

🎯 Project Goal

A platform that assists civilians during disasters by helping them navigate to safe zones, locate food distribution points, find nearby shelters, and send SOS alerts to rescue teams. Rescue workers can register, provide real-time help, and update resource locations. No login required for civilians.

⚠️ Problem Statement

During natural disasters, affected people lack real-time information on safe evacuation routes, food availability, and shelter locations. Rescue coordination is fragmented. This system creates a unified, accessible platform for both civilians and responders — without requiring civilian registration.

🛠️ Tech Stack

FrontendReact.js · Leaflet.js (Maps)
BackendNode.js · Express.js
DatabaseMongoDB (flexible for emergencies)
MapsOpenStreetMap / Google Maps API
AuthJWT (Rescue Team accounts only)
AlertsWebSockets / Push Notifications

🏗️ System Architecture

Civilian (No Login)
Rescue Team Portal
⬇ REST APIs + WebSocket
Node.js + Express.js Server
⬇ Queries
MongoDB
Maps API

📋 Planned Features

  • View safe evacuation routes on map
  • Find nearest food distribution points
  • Locate available shelters

  • No account/login needed
🔄 Rescue Team Features
